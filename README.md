# Shama Naz - Personal Resume Website

A modern, high-quality personal resume website built with **Next.js 15**, **TypeScript**, and **Tailwind CSS v4**.

## Features
- **Next.js 15 App Router**: Leveraging the latest React features.
- **Tailwind CSS v4**: Modern styling with CSS variables.
- **Framer Motion**: Smooth animations and section reveals.
- **Responsive Design**: Mobile-first approach.
- **Dark Mode**: Sleek dark theme with accent colors.
- **Print Optimized**: "Download CV" button that generates a clean PDF using browser print styles.

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## Folder Structure
- `app/`: Main application routes and global styles.
- `components/`: UI components for each section (Hero, Skills, Projects, etc.).
- `lib/`: Utility functions.
- `public/`: Static assets.

## Deployment
Deploy easily on [Vercel](https://vercel.com) by connecting your GitHub repository.
